AmneziaWG 2.0 — WireGuard с обфускацией для обхода блокировок в России

AmneziaWG — форк WireGuard с обфускацией пакетов, разработанный российской командой Amnezia для противодействия ТСПУ. В отличие от чистого WireGuard, который легко обнаруживается и замедляется в России, AmneziaWG рандомизирует заголовки и размеры пакетов — трафик становится неотличим от обычного QUIC-потока. Скорость и криптография — те же, что у WireGuard; добавляется только слой обфускации.

🔐 База: WireGuard + обфускация пакетов
🏗️ Разработчик: Amnezia (open-source)
📡 Статус РКН: ✓ Не обнаруживается ТСПУ

Что такое AmneziaWG и чем отличается от WireGuard

AmneziaWG — это протокол и семейство приложений, разработанных командой Amnezia (amnezia.org). Технически AmneziaWG — форк WireGuard, добавляющий параметры обфускации поверх стандартного WireGuard-туннеля. Протокол полностью совместим с WireGuard по криптографии (Curve25519, ChaCha20-Poly1305, BLAKE2s), но не совместим с обычными WireGuard-клиентами — для подключения нужно приложение Amnezia или другой клиент с поддержкой AmneziaWG.

Ключевые параметры обфускации, добавленные в AmneziaWG:

  • Junk packets (Jc, Jmin, Jmax) — перед установкой соединения клиент отправляет случайное количество мусорных пакетов случайного размера. Сигнатура инициации WireGuard (фиксированные 148 байт) «тонет» среди шума — DPI не может надёжно идентифицировать начало WireGuard-рукопожатия.
  • Init packet magic header (S1, S2) — случайные байты, добавляемые к заголовкам Initiation и Response пакетов. Фиксированные значения заголовков WireGuard (0x01, 0x02, 0x03, 0x04) заменяются на случайные, уникальные для каждого конфига.
  • Transport packet magic header (H1–H4) — рандомизация заголовков транспортных пакетов и keepalive. Весь трафик в установленной сессии теряет характерный WireGuard-паттерн.

Результат: ТСПУ, обнаруживающее WireGuard по сигнатурам 148/92-байтных пакетов с фиксированными заголовками, не способно применить те же правила к AmneziaWG — каждый конфиг имеет уникальные магические числа.

Как работает AmneziaWG

Обфускация на уровне пакетов

Стандартный WireGuard обнаруживается ТСПУ по трём характеристикам: фиксированным размерам пакетов инициации (148 байт) и ответа (92 байта), фиксированным 4-байтным заголовкам типов сообщений и поведенческому паттерну — редкие keepalive-пакеты между пакетами данных.

AmneziaWG ломает все три. Мусорные пакеты перед инициацией делают сигнатуру начала соединения непредсказуемой. Рандомизированные магические заголовки S1/S2/H1–H4 заменяют фиксированные значения типов сообщений. Итоговый трафик статистически напоминает QUIC-поток или случайный UDP — ТСПУ не имеет стабильной сигнатуры для детектирования.

Amnezia Free vs Amnezia Premium

🆓 Amnezia Free

Бесплатные серверы Amnezia. Работает через протокол AmneziaWG. Скорость ~8 Мбит/с. Открывает Instagram, WhatsApp, Telegram. Не работает для YouTube 4K и Discord из-за ограниченной скорости. Не требует регистрации — скачайте приложение и нажмите «Подключиться».

⭐ Amnezia Premium / Self-hosted

Полная скорость (до 91 Мбит/с). Собственный VPS или платный сервер Amnezia ($3.5/мес). Уникальный IP — не в базах блокировок коммерческих VPN. Открывает Instagram, YouTube 4K, Discord, WhatsApp. Kill Switch. Безлимит трафика.

Настройка собственного AmneziaWG-сервера на VPS

Собственный VPS с AmneziaWG — лучший вариант для постоянного использования: уникальный IP, нет ограничений скорости, полный контроль. Нужен VPS с Ubuntu 22.04+ (от $3–5/мес у Hetzner, DigitalOcean, Vultr).

Установка через скрипт Amnezia

Amnezia предоставляет скрипт автоматической установки, который разворачивает сервер AmneziaWG в Docker:

docker run -d --name amnezia-server \
  --restart unless-stopped \
  --privileged \
  -p 51820:51820/udp \
  -v amnezia-data:/opt/amnezia \
  amnezia/amnezia-server:latest

После запуска контейнера откройте приложение AmneziaVPN на телефоне или ПК → «+» → «Установить на своём сервере» → введите IP, логин и пароль VPS. Приложение автоматически подключится к серверу по SSH, установит Docker-контейнер и сгенерирует конфигурацию с AmneziaWG.

Параметры обфускации в конфигурации

Конфигурация AmneziaWG расширяет стандартный WireGuard-конфиг дополнительными параметрами в секции [Interface]:

[Interface]
PrivateKey = ПРИВАТНЫЙ_КЛЮЧ
Address = 10.8.0.2/32
DNS = 1.1.1.1
# Параметры AmneziaWG обфускации:
Jc = 4           # Количество мусорных пакетов (1–128)
Jmin = 40        # Минимальный размер мусорного пакета
Jmax = 70        # Максимальный размер мусорного пакета
S1 = 0           # Init magic header byte 1
S2 = 0           # Response magic header byte 1
H1 = 1           # Transport message type 1
H2 = 2           # Transport message type 2
H3 = 3           # Keepalive type
H4 = 4           # Cookie reply type

[Peer]
PublicKey = ПУБЛИЧНЫЙ_КЛЮЧ_СЕРВЕРА
Endpoint = IP_сервера:51820
AllowedIPs = 0.0.0.0/0, ::/0
PersistentKeepalive = 25

Для максимальной обфускации установите Jc = 4, Jmin = 40, Jmax = 70 и задайте уникальные значения H1–H4 (случайные числа вместо стандартных 1–4). Чем сильнее отличаются параметры от дефолтных WireGuard-значений — тем сложнее детектирование.

AmneziaWG vs VLESS+Reality — что выбрать?

Оба протокола не обнаруживаются ТСПУ и обеспечивают высокую скорость. Ключевые различия:

  • AmneziaWG проще в настройке: приложение Amnezia автоматически устанавливает сервер через SSH — не нужно понимать конфиги JSON или Xray. Для VLESS+Reality нужен Xray-core, панель 3X-UI или ручная настройка.
  • VLESS+Reality лучше маскируется: трафик имитирует TLS-соединение к конкретному доверенному домену. AmneziaWG выглядит как рандомный UDP-поток — менее «естественно», но не детектируется имеющимися сигнатурами.
  • AmneziaWG имеет Amnezia Free: бесплатный вариант без VPS для базового использования. У VLESS+Reality бесплатного аналога нет — только чужие конфиги из Telegram.
  • AmneziaWG нативный на всех платформах: официальное приложение Amnezia для iOS, Android, Windows, macOS без необходимости искать Shadowrocket или Hiddify.

Преимущества AmneziaWG

  • Простота «из коробки». Amnezia Free работает без VPS, без регистрации, без технических знаний — скачай и нажми «Подключиться».
  • Open-source с двумя аудитами безопасности. Код проверен независимыми исследователями, протокол прозрачен.
  • Нативный клиент для всех платформ. Одно приложение — Android, iOS, Windows, macOS, Linux. Нативный ARM на M-чипах.
  • Уникальный IP при self-hosted. Собственный VPS с AmneziaWG не входит в базы «известных VPN-IP» — меньше вероятности блокировки со стороны российских сервисов.

Недостатки

  • Amnezia Free медленный. ~8 Мбит/с — достаточно для Instagram и Telegram, но YouTube 4K недоступен. Для видео нужен Premium или VPS.
  • Требует VPS для полной скорости. В отличие от коммерческих VPN, AmneziaWG без VPS даёт только Free-серверы с ограниченной скоростью.
  • UDP-only. Как и WireGuard, AmneziaWG работает только по UDP — в сетях с жёсткой блокировкой UDP не работает.
  • Несовместим с обычными WireGuard-клиентами. Параметры Jc/Jmin/Jmax/S1/S2/H1–H4 понимают только клиенты с поддержкой AmneziaWG — стандартное приложение WireGuard их игнорирует.

Статус AmneziaWG в России — май 2026

AmneziaWG не обнаруживается и не замедляется ТСПУ по состоянию на май 2026 года. Это подтверждается стабильной работой на всех крупных операторах России — МТС, Мегафон, Ростелеком, Билайн. Рандомизированные заголовки и мусорные пакеты делают трафик статистически неотличимым от случайного UDP-потока.

Основной риск — не обнаружение протокола, а блокировка IP-адреса VPS. Если ваш VPS находится в популярных датацентрах (Hetzner, DigitalOcean, Vultr), его IP может периодически вноситься в реестры РКН. Решение: хостинг у менее известных провайдеров или оперативная смена IP при блокировке. Бесплатные серверы Amnezia Free используют IP с ротацией.

Сравнение с другими обфусцированными протоколами в 2026 году: AmneziaWG и VLESS+Reality — два лидирующих варианта по устойчивости к ТСПУ. Outline (Shadowsocks) работает, но детектируется сложным DPI на некоторых операторах. WireGuard без обфускации — замедляется.

Устранение неполадок AmneziaWG

Amnezia Free не подключается

  • Обновите приложение до последней версии — бесплатные серверы периодически меняются, старое приложение не знает новые адреса
  • Проверьте интернет-соединение: если нет базового доступа к сети, Amnezia не подключится
  • Попробуйте через мобильный интернет вместо Wi-Fi — некоторые домашние роутеры блокируют UDP на портах Amnezia

Собственный сервер не работает после установки

  • Убедитесь, что UDP-порт 51820 открыт на файрволе VPS: sudo ufw allow 51820/udp
  • Проверьте статус Docker-контейнера: docker ps → контейнер amnezia-server должен быть в статусе Up
  • Логи контейнера: docker logs amnezia-server — там будет причина ошибки
  • Убедитесь, что IP сервера не заблокирован РКН: с телефона на мобильном интернете попробуйте ping IP_сервера

Скорость ниже ожидаемой

  • На Amnezia Free скорость ограничена (~8 Мбит/с) — это нормально. Для высокой скорости нужен Premium или VPS
  • Выберите ближайший к России сервер: Финляндия, Германия, Нидерланды
  • Уменьшите Jc (количество мусорных пакетов) в конфиге — больше мусора = больше оверхед. Значение 4 оптимально
  • Проверьте CPU VPS при нагрузке: top — при >80% потребуется более мощный тариф

Перестало работать после обновления

  • После обновления приложения Amnezia иногда нужно переустановить конфигурацию — удалите и добавьте сервер заново
  • При обновлении Docker-контейнера на VPS — пересоздайте клиентскую конфигурацию через приложение
  • Проверьте, что параметры Jc/Jmin/Jmax/S1/S2/H1–H4 совпадают между клиентом и сервером
💡

AmneziaWG и роутеры Keenetic: AmneziaWG поддерживается на Keenetic через Entware. FAQ по AmneziaWG

AmneziaWG лучше обычного WireGuard?
Для России — однозначно да. WireGuard без обфускации замедляется ТСПУ на МТС и Мегафон. AmneziaWG добавляет обфускацию поверх WireGuard без изменения криптографии — скорость и безопасность те же, но ТСПУ не может детектировать трафик. Оверхед от обфускации минимален: несколько мусорных пакетов при инициации и несколько байт случайных заголовков — менее 1% от полезного трафика.
Amnezia Free чем ограничен?
Amnezia Free предоставляет бесплатные серверы с ограниченной скоростью (~8 Мбит/с). Это достаточно для Instagram, WhatsApp, Telegram и лёгкого серфинга. YouTube в 4K требует минимум 20–25 Мбит/с — Amnezia Free не обеспечивает. Discord голосовой чат и видеозвонки также могут быть нестабильны. Для YouTube 4K нужен Amnezia Premium ($3.5/мес) или собственный VPS (~$4/мес).
AmneziaWG совместим с обычным WireGuard?
По криптографии — да, совместим. Но дополнительные параметры (Jc, Jmin, Jmax, S1, S2, H1–H4) понимают только клиенты с поддержкой AmneziaWG: приложение Amnezia, форк WireGuard от Amnezia. Стандартный клиент WireGuard проигнорирует эти параметры и установит соединение как обычный WireGuard — без обфускации, которая и будет детектирована ТСПУ. Используйте именно приложение AmneziaVPN.
Как получить конфигурацию AmneziaWG без своего VPS?
Три варианта: (1) Amnezia Free — встроенные бесплатные серверы прямо в приложении, медленные (~8 Мбит/с); (2) Amnezia Premium — платные серверы от команды Amnezia ($3.5/мес, быстрые); (3) знакомый с VPS может поделиться конфигурацией AmneziaWG через приложение Amnezia (кнопка «Поделиться»). Стороннее «добавить AmneziaWG-конфиг как обычный WireGuard» — не работает без обфускации.
