Hiddify — универсальный прокси-клиент для обхода блокировок в России

Hiddify — это не протокол, а бесплатное кросс-платформенное клиентское приложение на базе движка sing-box. Работает на Android, iOS, Windows, macOS и Linux. Поддерживает практически все современные протоколы обхода блокировок: VLESS+Reality, VMess, Trojan, Shadowsocks, Hysteria2, TUIC и другие. Главное отличие от конкурентов (v2rayN, Shadowrocket) — современный интерфейс, автоматический выбор сервера с минимальным пингом и единый клиент для всех платформ.

Что такое Hiddify — клиент, а не протокол

Важно понять с самого начала: Hiddify — это приложение-клиент, а не самостоятельный протокол. Это как разница между Firefox (браузер) и HTTP (протокол). Hiddify — оболочка, которая умеет говорить на десятках протоколов. Сами протоколы — VLESS, VMess, Shadowsocks, Hysteria2 — описаны в отдельных руководствах этого сайта.

Hiddify создан иранской командой разработчиков (GitHub: hiddify/hiddify-app) как ответ на жёсткую интернет-цензуру в Иране и постепенно завоевал популярность в России, Китае и других странах с блокировками. Приложение полностью open-source, без рекламы и без встроенных серверов — пользователь сам предоставляет конфигурацию своего сервера или подписку.

Экосистема Hiddify состоит из двух компонентов:

  • Hiddify (клиент) — приложение для конечных пользователей. Принимает ссылки-конфиги или URL-подписки, автоматически выбирает лучший сервер, создаёт VPN-туннель. Доступно для Android, iOS, Windows, macOS и Linux.
  • Hiddify Manager — серверная веб-панель для администраторов. Устанавливается на VPS, позволяет создавать пользователей, управлять подписками и настраивать поддерживаемые протоколы. Автоматически генерирует URL-подписки для Hiddify-клиентов.

Как работает Hiddify

Ядро sing-box

Под капотом Hiddify работает на sing-box — универсальном прокси-движке нового поколения, написанном на Go. sing-box пришёл на смену разрозненным реализациям (shadowsocks-libev, xray-core, hysteria) и объединил их в одном ядре. Это означает, что Hiddify поддерживает любой протокол, который поддерживает sing-box: VLESS, VMess, Trojan, Shadowsocks, Hysteria2, TUIC, ShadowTLS, SSH, XHTTP, Psiphon, Naive, Mieru.

sing-box создаёт системный TUN-интерфейс (tun0) — весь трафик устройства проходит через него. Это означает, что в отличие от SOCKS5-прокси, Hiddify защищает не только браузер, но и все приложения: Telegram, WhatsApp, игры, системные сервисы.

Подписки и автовыбор сервера

Ключевая особенность Hiddify — работа с URL-подписками. Подписка — это ссылка вида https://…/sub, по которой приложение загружает список серверов в формате Xray, Clash или sing-box. При обновлении подписки список автоматически обновляется — новые серверы добавляются, заблокированные исчезают без участия пользователя.

Функция автовыбора сервера измеряет пинг до каждого узла подписки и автоматически переключается на наиболее быстрый. Это особенно ценно в России, где разные серверы в разное время могут замедляться ТСПУ — Hiddify сам находит рабочий.

Режимы работы

  • Global Mode — весь трафик устройства через прокси
  • Rule-based Mode — умная маршрутизация: российские сайты напрямую, заблокированные через прокси. Аналог пресета «Россия» в v2rayN
  • Direct Mode — отключить прокси (VPN выключен)

Для России рекомендуется Rule-based Mode — российские банки, Яндекс и Госуслуги открываются напрямую, Instagram, YouTube и Discord — через прокси.

Настройка Hiddify на устройствах

Для начала работы с Hiddify нужен источник конфигурации: URL-подписка от своего сервера (Hiddify Manager, 3X-UI, Marzban) или от провайдера, либо отдельные ссылки-конфиги вида vless://…, vmess://…, ss://…. Бесплатные конфиги можно найти в Telegram-каналах — подробнее в разделе V2Ray / Xray.

Android — Hiddify

  1. Установите Hiddify из Google Play или APK с github.com/hiddify/hiddify-app/releases
  2. При первом запуске приложение предложит добавить профиль. Нажмите «Добавить профиль»
  3. Выберите способ добавления: «Из ссылки» (URL-подписка), «Из буфера» (конфиг-ссылка vless://…) или «Сканировать QR»
  4. После добавления профиля Hiddify загрузит список серверов и покажет их с пингом
  5. Выберите режим Rule-based в нижнем меню (значок воронки)
  6. Нажмите большую кнопку подключения в центре экрана → разрешите VPN-запрос
  7. Обязательно: Настройки Android → Приложения → Hiddify → Батарея → Без ограничений. Иначе Android завершит приложение в фоне
📱

iPhone — Hiddify Proxy & VPN

  1. Установите Hiddify Proxy & VPN из App Store (бесплатно)
  2. Нажмите «+» → «Add Profile»
  3. Вставьте URL-подписку или ссылку-конфиг → нажмите «Import»
  4. На главном экране нажмите «Connect» → разрешите «Добавить конфигурации VPN»
  5. Выберите режим работы: Rule-based для России (банки напрямую)
  6. Отключите iCloud Private Relay: Настройки → Apple ID → iCloud → Private Relay → Выкл
🖥️

Windows — Hiddify

  1. Скачайте Hiddify с github.com/hiddify/hiddify-app/releases → файл Hiddify-Windows-Setup-x64.exe или из Microsoft Store
  2. Установите → запустите от имени администратора при первом запуске
  3. Нажмите «+» → «Add Profile from URL» → вставьте URL-подписку или конфиг-ссылку
  4. После загрузки профиля в списке появятся серверы с пингом
  5. Выберите режим «Rule-based» — в этом режиме ru-домены и российские IP идут напрямую
  6. Нажмите кнопку подключения. При запросе — разрешите сетевой адаптер (UAC)
  7. Добавьте папку Hiddify в исключения антивируса
  8. Проверьте: ipleak.net → нероссийский IP
🍎

macOS — Hiddify

  1. Скачайте Hiddify с github.com/hiddify/hiddify-app/releases → Hiddify-MacOS.dmg
  2. Перетащите в «Программы» → запустите → разрешите сетевое расширение в Системных настройках → Конфиденциальность
  3. Нажмите «+» → «Add Profile» → вставьте URL или конфиг-ссылку
  4. Выберите режим Rule-based
  5. Нажмите «Connect» — иконка в Menu Bar сменится на активную
  6. Отключите iCloud Private Relay
  7. Для автозапуска: добавьте Hiddify в Системные настройки → Основные → Объекты входа

Hiddify Manager — серверная панель

Если хочется иметь собственный VPN-сервер с удобным управлением, Hiddify Manager — один из лучших вариантов в 2026 году. Это веб-панель, которая устанавливается на VPS и автоматически настраивает Xray/sing-box с выбранными протоколами. Главные преимущества перед ручной настройкой конфига: веб-интерфейс, управление пользователями, автоматическая генерация подписок и поддержка нескольких протоколов одновременно.

Установка Hiddify Manager на Linux VPS

Нужен VPS с Ubuntu 22.04 или 24.04, минимум 512 МБ RAM. Установка одной командой:

bash -c "$(curl -Ls https://raw.githubusercontent.com/hiddify/hiddify-manager/main/common/download_install.sh)"

Скрипт установит Hiddify Manager с автоматической настройкой HTTPS через Let's Encrypt. После завершения (5–10 минут) в терминале появится URL панели администратора вида https://IP_сервера/ваш_путь/. Войдите, смените пароль и настройте нужные протоколы.

Что делать после установки

  1. Откройте панель администратора → раздел «Protocols» → включите нужные протоколы (рекомендуется: VLESS+Reality + Shadowsocks как резерв)
  2. Раздел «Users» → создайте пользователя → скопируйте URL-подписку
  3. Вставьте URL-подписку в Hiddify-клиент на телефоне или ПК → все серверы загрузятся автоматически
  4. Поделитесь подпиской с семьёй или друзьями — каждый получит актуальный список серверов

Hiddify Manager конкурирует с 3X-UI и Marzban как серверная панель. Главное отличие: Hiddify Manager более «вертикальная» — меньше настроек, но быстрее настраивается и лучше интегрирована с клиентским приложением Hiddify. 3X-UI гибче, но сложнее. Подробнее о настройке Xray-сервера — в разделе V2Ray / Xray.

Когда выбирать Hiddify

  • Нужен единый клиент для всех устройств. Один и тот же интерфейс и одна подписка работает на Android, iPhone, Windows, macOS и Linux — нет необходимости изучать разные приложения под каждую платформу.
  • Нужна автоматическая маршрутизация для России. Rule-based режим Hiddify «из коробки» отправляет российские сайты напрямую, заблокированные — через прокси, без ручной настройки правил.
  • Используете Hysteria2 или TUIC. Эти новые высокоскоростные протоколы (работают по QUIC/UDP) поддерживаются Hiddify, но не v2rayN/v2rayNG без специальных плагинов.
  • Хотите свой сервер с удобной панелью. Связка Hiddify Manager (сервер) + Hiddify клиент — одно из самых простых self-hosted решений в 2026 году.

Ограничения Hiddify

  • Не подходит как замена коммерческому VPN. Hiddify не предоставляет серверы — нужен либо собственный VPS, либо подписка от провайдера или сообщества. Без источника конфигурации приложение само по себе ничего не делает.
  • sing-box ядро потребляет больше батареи, чем Outline. Некоторые пользователи замечают повышенный расход на Android — особенно на устройствах Huawei и старых Samsung. Отключение батарейных ограничений обязательно.
  • Нет собственного Kill Switch уровня ОС на Windows. TUN-режим sing-box обеспечивает защиту, но при внезапном сбое трафик теоретически может пойти напрямую. Для критичных сценариев дополняйте системным файрволом.

Устранение неполадок Hiddify

Приложение подключается, но сайты не открываются

  • Убедитесь, что выбран режим Global или Rule-based, а не Direct
  • Проверьте ipleak.net — если российский IP, TUN-туннель не работает. Перезапустите приложение от имени администратора (Windows)
  • На macOS разрешите сетевое расширение в Системных настройках → Конфиденциальность → убедитесь, что расширение Hiddify не заблокировано
  • Отключите iCloud Private Relay на iOS/macOS — он конфликтует с TUN-режимом

Hiddify работает, но периодически обрывается на Android

  • Главная причина: Android завершает процесс для экономии батареи
  • Настройки → Приложения → Hiddify → Батарея → Без ограничений
  • На Samsung: «Уход за устройством» → Батарея → удалите Hiddify из спящего режима
  • На Huawei/Honor: Настройки → Приложения → Hiddify → Автозапуск → разрешить
  • Включите «Автоподключение при запуске» в настройках Hiddify

Не загружается подписка — «Profile update failed»

  • URL-подписка может быть временно недоступна — попробуйте обновить позже
  • Некоторые URL-подписки требуют предварительного подключения — попробуйте добавить вручную один конфиг (vless://…), подключиться, затем обновить подписку
  • Проверьте, что URL скопирован целиком без лишних пробелов
  • Если Hiddify Manager свой — проверьте, что сервер работает и SSL-сертификат актуален

Некоторые сайты в Rule-based режиме всё равно открываются через прокси

  • Правила маршрутизации в Hiddify обновляются с обновлением приложения — обновитесь до последней версии
  • Конкретные домены можно добавить в исключения: Настройки → Routing → Bypass → добавьте домен
  • Для банков с проблемами добавьте их домены в «Direct» список явно

FAQ по Hiddify

Hiddify — это VPN или прокси?
Hiddify — клиентское приложение, которое может работать как в режиме VPN (TUN-интерфейс, весь трафик), так и в режиме прокси (только выбранные приложения). Это не протокол сам по себе — Hiddify использует протоколы других авторов: VLESS, Shadowsocks, Hysteria2 и другие. Движок — sing-box, поддерживающий практически все современные протоколы обхода блокировок.
Hiddify бесплатный?
Да, полностью бесплатный и без рекламы. Клиентское приложение Hiddify (для Android, iOS, Windows, macOS, Linux) — бесплатно с открытым кодом. Hiddify Manager (серверная панель) — тоже бесплатно для некоммерческого использования. Платить придётся только за VPS-сервер (от $3–5/мес), если хотите свой сервер.
В чём разница между Hiddify и v2rayN/v2rayNG?
Оба являются клиентами для работы с протоколами V2Ray/Xray. Главные отличия Hiddify: современный удобный интерфейс, поддержка новых протоколов (Hysteria2, TUIC) через sing-box, единое приложение для всех платформ включая iOS. v2rayN — более зрелое приложение для Windows с большим числом настроек, популярен у продвинутых пользователей. v2rayNG — аналог для Android. Для начинающих Hiddify проще; для тонкой настройки — v2rayN гибче.
Нужен ли свой сервер для Hiddify?
Нет, но нужен источник конфигурации. Варианты: (1) Собственный VPS с Hiddify Manager или 3X-UI — наиболее надёжный вариант; (2) конфиги от знакомого, кто уже настроил сервер; (3) бесплатные подписки из Telegram-каналов — работают, но нестабильны и небезопасны для приватной информации; (4) коммерческий провайдер, поддерживающий V2Ray/Xray подписки.
Работает ли Hiddify в России в 2026 году?
Hiddify как приложение работает без ограничений. Устойчивость к ТСПУ зависит от выбранного протокола в конфиге: VLESS+Reality — не блокируется, надёжный выбор для России. Shadowsocks — работает с оговорками. WireGuard — замедляется. Подбирайте протокол исходя из статусов в наших разделах по каждому протоколу.
